Welcome to the model_DC-Power_controller wiki!

this was made for a Display Model Railroad. that needed automation with just a push of a button. It when activated will turn on relays for display lights ramp up a dc power supply from 0 to 100% of power and run for 2 minutes then ramp down to 0% of power and then turn off the relays and the wait nor the next cycle. was made to be customizable and expandable to many different setups, and does alternate modes for testing and safety.
